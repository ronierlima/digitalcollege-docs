# Descrição da atividade

Você deve criar um componente de tabela simples usando React. Esse componente deve receber um array de objetos como entrada, onde cada objeto representa uma linha na tabela. Cada objeto deve ter as chaves id, nome, usuario e email, e o valor dessas chaves deve ser exibido em células correspondentes na tabela.

O componente deve ter as seguintes características:

1. Ser capaz de exibir as linhas da tabela com base nos dados do array de objetos que são passados como propriedade. As colunas da tabela devem ser as seguintes: id, nome, usuário e email.

```js
const dados = [
  { id: 1, nome: "João", usuario: "joao123", email: "joao123@email.com" },
  { id: 2, nome: "Maria", usuario: "maria456", email: "maria456@email.com" },
  { id: 3, nome: "Pedro", usuario: "pedro789", email: "pedro789@email.com" },
];
```

2. O componente deve ser capaz de renderizar corretamente uma tabela mesmo se o array de objetos for vazio.

3. O layout e o estilo da tabela são de sua escolha, mas devem ser legíveis e esteticamente agradáveis.

4. Ao concluir a atividade, você deve demonstrar o componente de tabela funcionando em um aplicativo React, mostrando uma tabela com as linhas e colunas corretas exibidas a partir do array de objetos que você passou como entrada. Seu código deve estar limpo, organizado e documentado.
