
# Descrição da atividade

1. Finalize a estilização da tela de login de acordo com a sua preferência, utilizando CSS.

```js

  const corpo = {  email: "joao123@email.com", password: "joao123"};

```
2. Complete o método "autenticar" para enviar uma requisição utilizando o verbo HTTP POST para a API "https://usuarios.ronierlima.dev/auth". O corpo da requisição deve ser um objeto com as propriedades "email" e "password", contendo os valores inseridos nos campos de email e senha do formulário.
3. Armazene as informações do usuário retornado pela API em um estado do componente.
4. Apresente o nome do usuário em um elemento `<h1>` na mesma tela de login.
5. Crie uma página de cadastro de usuário com os seguintes campos no corpo da requisição: "email", "name" e "password". Implemente a lógica necessária para enviar uma requisição HTTP POST para a API "https://usuarios.ronierlima.dev/register" ao submeter o formulário de cadastro.
