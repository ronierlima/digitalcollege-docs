
# Descrição da atividade

0 - Faça um fork do projeto https://github.com/ronierlima/digitalcollege-react e crie uma nova branch com o seu nome, por exemplo: 'bugfix/seu_nome'. Esse processo é similar ao que foi feito na prática de pull request.

1 - No código fornecido, a tabela de cidades não é atualizada quando um estado é selecionado no menu suspenso. Identifique se há algo faltando no código e priorize o uso dos efeitos colaterais.

2 - Na raiz do projeto, crie um arquivo com o seu nome, como por exemplo: 'README_SEU_NOME.md'. Nesse arquivo, descreva partes do código que você não entendeu o que fazem e explique o motivo, se é porque é algo novo para você ou porque não se lembra mais do que a função realiza.

3 - Por fim, abra um pull request com suas mudanças e aguarde a avaliação.
