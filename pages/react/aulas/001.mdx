# Usei o CRA, e agora ?

Quando você executa o comando npx create-react-app my-app, o Create React App cria uma estrutura básica de arquivos para você começar a desenvolver sua aplicação React. Aqui está uma lista dos principais arquivos e pastas gerados:

### **node_modules**: 

Esta pasta contém as dependências do seu projeto instaladas pelo npm ou yarn.

###  **public**: 

Esta pasta contém os arquivos públicos que serão servidos pelo servidor web. É aqui que você coloca o arquivo HTML principal e outros arquivos estáticos, como imagens e fontes.
    
    - **index.html**: Este é o arquivo HTML principal que é servido pelo servidor web. É aqui que você inclui os scripts React e outros arquivos estáticos que sua aplicação precisa.
    - **favicon.ico**: Este é o ícone da aplicação que aparece na aba do navegador.

### src:

Esta pasta contém o código-fonte da sua aplicação React.

- **App.js**: Este é o componente principal da sua aplicação React. Ele é importado e renderizado no arquivo index.js.
- **index.js**: Este arquivo é o ponto de entrada da sua aplicação React. Ele importa e renderiza o componente App.
- **index.css**: Este arquivo contém o estilo global da sua aplicação.
- **logo.svg**: Este é o logotipo React padrão, que é exibido na tela inicial do seu aplicativo React.
- **serviceWorker.js**: Este arquivo contém o código para o Service Worker, que é uma tecnologia usada para criar aplicativos da Web progressivos (PWA).

### package.json: 

Este arquivo contém as informações sobre o seu projeto, como nome, versão, dependências e scripts de execução.

### package-lock.json: 

Este arquivo contém as informações sobre as dependências instaladas pelo npm ou yarn.

### README.md: 

Este arquivo é uma breve descrição do seu projeto e como executá-lo 

### Além disso

Além disso, o Create React App configura um ambiente de desenvolvimento completo com ferramentas como o Webpack, Babel e o servidor de desenvolvimento interno do React. Essas ferramentas permitem que você escreva código em ES6 e JSX, use recursos avançados do JavaScript, como async/await, e testar e depurar seu aplicativo de forma eficiente

Em resumo, o Create React App cria uma estrutura básica de arquivos e configura um ambiente de desenvolvimento completo para você começar a construir sua aplicação React. Você pode personalizar e expandir essa estrutura de arquivos de acordo com as necessidades do seu projeto.
